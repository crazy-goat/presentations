# Crazy Goat Software - Presentations

## Prepare

```shell
make build
```

## Run

```shell
make start
```

visit http://127.0.0.1:8000

## Run development

```shell
npm start -- --port=8001
```
visit http://127.0.0.1:8001

### Keys

- `s` - speaker view
- `f` - full screen
- `esc` - slides overview
